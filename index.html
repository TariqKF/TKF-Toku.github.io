Projects

01
Academic Project 2025

Electronic Speed Controller (ESC)

Designed a custom 24 V DC motor Electronic Speed Controller (ESC) supporting up to 5 A of continuous current. The project prioritized safe power delivery, high-efficiency switching, and precise motor speed control. The board layout was optimized for thermal dissipation and trace widths appropriate for the 5 A current draw.

Engineering Highlights

  • Designed multi-layer PCB layouts in KiCad featuring dedicated ground planes and power pours.
  • Implemented via stitching to improve ground plane path efficiency and thermal conductivity.
  • Selected components by cross-referencing datasheets for tolerance, footprint size, and availability.
  • Styled trace geometry to withstand high-current thresholds safely without overheating.
  • Configured PWM signal input routes for motor speed throttling.

Personal Project 2026

Embedded MCU Power & Programming Board

Designed a custom microcontroller (MCU) development board that accepts high-voltage power input (24-36 V) and steps it down to a clean, stable 3.3 V using an onboard buck regulator. The board incorporates a modern USB-C connector for programming and board power, along with necessary protective circuitry.

Engineering Highlights

  • Integrated a buck regulator circuit to step down 24-36 V voltage sources to a stable 3.3 V for embedded hardware.
  • Configured USB-C connector routes for dual-mode power delivery and serial flashing interface.
  • Designed protection support circuitry, including ESD protection and reverse polarity prevention.
  • Optimized PCB layout around regulator decoupling capacitors to minimize high-frequency noise.
  • Planned clear ground routing to avoid interference with analog signals.

Remote Control Robot
